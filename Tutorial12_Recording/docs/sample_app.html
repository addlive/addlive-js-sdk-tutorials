Recording sample app description

After opening the sample app website we see following website:

We have to fill in "App ID, "API key", "Scope ID" fields and optionally change "User ID" (if given User ID is already present in session to be recorded). After pressing "REC" button, recording starts:

After stopping recording with "STOP" button, new status is displayed:

In this moment we can poll status of the upload and transcoding using "STATUS" button. When everything is ready, "DESCRIPTION" button will be available and following status will be displayed:

Pressing "DESCRIPTION" button fills in activity table with events together with media links

"open" link allows for media preview or download.
